Cint es una plataforma web cuyo propósito es conectar a pequeños y medianos productores nicaragüenses directamente con consumidores y empresas. Nuestro objetivo principal es eliminar intermediarios innecesarios, garantizando precios justos para todos los involucrados.

## Descripción del Proyecto

A diferencia de los e-commerce tradicionales enfocados en el revendedor, esta aplicación pone en el centro al **productor**. La plataforma permite a los creadores de productos agrícolas y artesanales establecer y justificar el precio real de su trabajo, garantizando que sus esfuerzos sean valorados monetariamente.

## Instalación

### Requisitos del sistema
Para ejecutar este proyecto en un entorno local de desarrollo, necesitas tener instalados los siguientes componentes:

* **[Node.js](https://nodejs.org/)** (v16.0 o superior) y **npm** (para el frontend en React.js).
* **[.NET 8.0 SDK](https://dotnet.microsoft.com/)** (o la versión correspondiente al backend).
* **[SQL Server](https://www.microsoft.com/sql-server/)** (junto con SQL Server Management Studio o Azure Data Studio).
* Cuenta activa en **[Firebase](https://firebase.google.com/)** (para Autenticación y base de datos en tiempo real) y **[Google Cloud](https://cloud.google.com/)** (para las APIs de Maps y Places).
